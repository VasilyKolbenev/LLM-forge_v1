Agent Framework

Built-in ReAct agent + LangGraph, CrewAI, AutoGen. Choose your framework per node in the same pipeline.

Guardrails

PII detection/masking, prompt injection defense, toxicity filtering, JSON schema validation. Block, warn, or mask.

MCP + A2A Protocols

Expose tools via MCP (Claude, Cursor compatible). Delegate tasks to remote agents via Google's A2A protocol.

Experiment Tracking

Local, ClearML, or W&B backends. Dataset fingerprinting, environment capture, HPO sweeps with Optuna.

Automated evaluation with customizable criteria. Traffic splitting, statistical winner detection, canary deployment.

Deploy in 60 Seconds

Three ways to get started.

🐍 pip install

pip install llm-forge[ui]
forge ui
# → http://localhost:8888

🐳 Docker

docker compose up -d
# GPU passthrough included
# → http://localhost:8888

👨‍💻 From Source

git clone ...llm-forge
pip install -e ".[ui,dev]"
cd ui && npm run build
forge ui
